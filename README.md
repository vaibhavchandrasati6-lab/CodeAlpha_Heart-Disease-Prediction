# ❤️ Heart Disease Risk Predictor

🚀 This project was developed during my **Machine Learning Internship at CodeAlpha** (CodeAlpha), where I worked on building real-world, end-to-end machine learning solutions.

---

## 💼 Internship Context & My Contribution

As part of this internship, I was responsible for designing and implementing a **complete machine learning pipeline** for predicting heart disease risk.

### 🔹 Key Responsibilities:

* Conducted **Exploratory Data Analysis (EDA)** to understand patterns and correlations in patient health data
* Performed **data preprocessing**, including handling missing values, encoding categorical variables, and feature scaling
* Built and compared multiple models:

  * Logistic Regression
  * Random Forest
  * Support Vector Machine (SVM)

* Deployed the final model as an **interactive Streamlit web application**

## 📌 Project Overview

The application predicts whether a patient is at **high risk** or **low risk** of heart disease based on medical attributes such as age, cholesterol level, blood pressure, and more.

It provides:

* ✅ Risk classification
* 📊 Probability score for better interpretation
